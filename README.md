# PHP CRUD Project

This is a simple PHP CRUD project connected with MySQL database.

## Features
- Insert data
- Show data
- Update data
- Delete data

## Technologies Used
- PHP
- MySQL
- HTML
- XAMPP

## Database
Table name: `emp`

Fields:
- id
- name
- address

## How to Run
1. Start Apache and MySQL in XAMPP
2. Create database in phpMyAdmin
3. Create `emp` table
4. Place project folder inside `htdocs`
5. Run the project in browser using localhost

## Project Purpose
This project was made for practice to understand CRUD operations in PHP and MySQL.
